OSFamilyテーブル要件

識別情報
id: BIGSERIAL, 主キー
name: VARCHAR(30), NOT NULL, UNIQUE

説明
マスターデータ（OS ファミリー参照テーブル）
OS の大分類を標準化。
例: Windows, Linux, macOS など

制約
name は NULL 不可、UNIQUE
OS ファミリー名の一意性

推奨インデックス
INDEX name（UNIQUE に含まれる）

PostgreSQL DDLサンプル

CREATE TABLE os_family (
  id BIGSERIAL PRIMARY KEY,
  name VARCHAR(30) NOT NULL UNIQUE
);
